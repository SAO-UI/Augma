| 参数    | 说明         | 类型    | 可选值                                                | 默认值  |
| ------- | ------------ | ------- | ----------------------------------------------------- | ------- |
| type    | 类型         | string  | default / primary / success / warning / danger / info | default |
| icon    | 是否图标按钮 | boolean | -                                                     | false   |
| outline | 是否描边按钮 | boolean | -                                                     | false   |
